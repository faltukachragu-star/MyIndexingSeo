Complete API Setup Guide

1

Create a Google Cloud Project

  1. Go to the Google Cloud Platform Console.
  2. Give your project a name (e.g., "Website Indexing") and click Create.
2

Enable the Indexing API

  1. Go to the Indexing API Page (make sure your project is selected).
  2. Click the blue Enable button.
3

Create a Service Account

  1. Go to the Service Accounts page.
  2. Click + Create Service Account.
  3. Give it a name (e.g., "indexing-service-bot") and click Create and Continue.
  4. For Role, select Owner. Click Continue, then click Done.

    5. Copy the new service account's email address. You will need it in the next step. It looks like: name@project-id.iam.gserviceaccount.com.

4

Connect to Google Search Console

This is the most important step. You must tell Search Console that your new Service Account is an owner of your website.

  1. Go to your Google Search Console and select your website.
  2. Go to Settings → Users and permissions.
  3. Click the blue Add User button.
  4. In the "Email address" field, paste the service account email you copied in Step 3.
  5. Set the Permission to Owner. Click Add.
5

Download Your JSON Private Key

  1. Go back to the Google Cloud Service Accounts page.
  2. Click on the service account you created.
  3. Go to the KEYS tab, click Add Key → Create new key.
  4. Choose JSON and click Create. A file will download. This is your API key!

You now have everything you need. Go back to the tool and upload this key file to begin indexing.
