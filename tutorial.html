Rapid Indexer

Complete API Setup Guide

1

Create a Google Cloud Project

  1. Go to the Google Cloud Platform Console.
  2. Give your project a name (e.g., "Website Indexing") and click Create.
2

Enable the Indexing API

  1. Go to the Indexing API Page.
  2. Make sure your project is selected at the top, then click Enable.
3

Create a Service Account

  1. Go to the Service Accounts page.
  2. Click + Create Service Account. Give it a name, click Create and Continue.
  3. For Role, select Owner. Click Continue, then Done.

Copy the new service account's email address. You need it next!

4

Connect to Google Search Console

  1. Go to your Google Search Console.
  2. Go to Settings → Users and permissions.
  3. Click Add User, paste the service account email, and set Permission to Owner. Click Add.
5

Download Your JSON Key

  1. Go back to the Google Cloud Service Accounts page and click on your new account.
  2. Go to the KEYS tab, click Add Key → Create new key.
  3. Choose JSON and click Create. A file will download. This is your API key.
